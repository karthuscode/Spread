Spin Flow

This document defines the order of events during a base game spin.

Spin Sequence

1. The player starts a spin.
2. The selected bet is deducted from the balance.
3. The game generates the initial 5×3 symbol grid.
4. Naturally generated Virus Wild symbols are identified.
5. Each Virus Wild checks its directly adjacent positions.
6. Eligible Human symbols are transformed into Wild symbols.
7. The final grid is created.
8. The game evaluates every active payline from left to right.
9. The Virus Wild substitutes for regular paying symbols, including the Masked Person.
10. The highest valid symbol combination is selected on each payline.
11. All payline wins are added together.
12. The total win is added to the player’s balance.

Important Rule

Paylines are evaluated only after the infection mechanic has finished.

Newly created Wild symbols do not spread the infection further in the base game.

Current Scope

The current base game does not include:

* chain reactions;
* bonus features;
* free spins;
* cascading symbols;
* multipliers;
* right-to-left wins.
