Infection Mechanic

Overview

The main mechanic of Spread is based on a Virus Wild symbol that can infect nearby human symbols.

When a human symbol becomes infected, it transforms into a Wild symbol and can help create winning combinations.

Grid

The game uses a 5×3 grid.

The grid contains:

* 5 columns
* 3 rows
* 15 symbol positions

	Virus Wild

The Virus Wild is the main special symbol of the game.

It can:

* substitute for regular paying symbols;
* infect nearby unprotected human symbols;
* transform infected humans into Wild symbols.

Adjacent Symbols

The Virus Wild checks the four directly adjacent positions:

* above;
* below;
* left;
* right.

Diagonal positions are not considered adjacent.

Example:

    X
X   V   X
    X

V represents the Virus Wild.

X represents positions that can potentially be infected.

Infectable Symbols

The Virus Wild can infect regular human symbols.

An infected human:

1. changes into a Wild symbol;
2. becomes part of the final game grid;
3. can contribute to winning combinations.

Protected Symbols

The Masked Person is immune to infection.

The Virus Wild cannot infect:

* Masked Person symbols;
* tool symbols;
* other Virus Wild symbols.

Base Game Limitation

In the base game, newly created Wild symbols do not spread the infection further.

Only the Virus Wild symbols that originally appeared on the grid can infect adjacent humans.

Current Scope

The current prototype includes:

* a 5×3 grid;
* regular human symbols;
* a Masked Person symbol;
* tool symbols;
* the Virus Wild;
* one-step infection.

The current prototype does not include:

* chain reactions;
* free spins;
* an Outbreak bonus;
* multipliers;
* cascading symbols.
