Symbols

The current version of the game contains the following symbol categories.

Human

Regular human symbols can be infected by the Virus Wild.

The game should include at least three different human designs to create more visual variety.

Masked Person

The Masked Person is a high-value symbol that is immune to infection.

It cannot be transformed into a Wild symbol by the Virus Wild.

Medical Equipment

Medical equipment symbols are currently planned as low-value symbols.

The game should include at least three different equipment designs to create more visual variety.

The exact equipment symbols have not been confirmed yet.
