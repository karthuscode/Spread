Paytable Draft

Overview

This document contains the initial symbol weights and payout values for the Spread prototype.

All values are experimental and must be validated through simulation.

Bet Model

The game uses five fixed paylines.

The total bet is divided equally between all paylines.

Example:

Total bet: 1 credit
Paylines: 5
Bet per line: 0.2 credits

Wild Rules

The Virus Wild has no separate payout.

It substitutes for every regular paying symbol, including the Masked Surgeon.

An all-Wild payline is evaluated as the regular symbol combination that produces the highest payout.

Validation Goals

The initial simulation should measure:

* total RTP;
* hit frequency;
* average win;
* Virus Wild frequency;
* infection frequency;
* average number of infected symbols per spin.

The target RTP for the first prototype is approximately 94%–97%.

This target is experimental and does not represent a certified commercial game.
